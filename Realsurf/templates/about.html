REALSURF is a personal project by David Owens, a student at the University of California, San Diego. The purpose of this project is to gain experience with web developement. This site is built upon the Python Flask framework and utlizes the BeautifulSoup web scraping library to obtain the surf data displayed. This site serves as a "proof of concept" hence the small collection of surf breaks accessible to the user. So what makes this site different than others? Well first it's not a crowded mess and will never have ads, but also the surf data displayed are all calcuated from a weighted average based upon multiple other surf report websites. Hopefully, by doing this calculuation, the user will be provided with the most accurate surf report possible, leading to better decision making. If you encounter a bug, please let me know at davidowens088@gmail.com. Legal: All data obtained from other report sites like surfline.com, magicseaweed.com is rightfully owned by the respective site. REALSURF is not and never will be intended for profit/commercial use.
